03 / Project format

Transparent and readable.
No hidden asset library.

An exported project is simply JSON configuration plus one complete PNG—easy to back up and ready for other tools.

Codex v2 compatible

Keep it simple. Keep it free.

pet.json describes identity and atlas location, spritesheet.png holds every frame, and studio.json records richer animation and trigger logic.

Read the format docs ↗ 
my-pet/
├── pet.json          # identity and atlas path
├── spritesheet.png   # the complete atlas
└── studio.json       # animation and event logic
